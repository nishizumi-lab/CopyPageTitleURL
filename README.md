# Get PageTitle & URL

## What's this?
This is a Chrome extension that lets you grab the title and URL of a webpage and copy it in plain text, HTML, or Markdown format.
これは、Webページのタイトルと URL を取得し、Markdown形式でコピーできる Chrome 拡張機能です。
